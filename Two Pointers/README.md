## Two Pointers

- Two pointers is a common programming technique to solve coding questions that involves an iterable data structure (e.g. arrays/string), specially for interview coding assessment.
- Given the nature of the problem, two pointers can be implemented in different ways.

## Classification
- Same Directions: Two pointers moving forward in same directions.
- Opposite Directions: Generally Two pointers moving towards the center of data structure one from the beginning and other form the end of the array/string. 


## Note:
- Sometimes we could use index from the loop as one of the pointers. This way Two pointers doesn't have to be solved using two pointers.
